Implementation of graph neural network architectures for weather forecasting, including DeepMind's Functional Generative Network (FGN), GenCast, WindBorne's WeatherMesh-3, Microsoft's Aurora, and FengWu-GHR. The project provides modular components for graph-based weather prediction, enabling experimentation with different GNN architectures on global weather data.
