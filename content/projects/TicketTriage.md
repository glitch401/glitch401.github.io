A high-performance local AI agent that triages customer support tickets, validates order details, and drafts policy-compliant responses. Built with LangGraph for stateful conversation management using "Sticky Threads" that bind conversation history to Order IDs. Includes full observability via LangSmith tracing and a Next.js chat UI frontend.
