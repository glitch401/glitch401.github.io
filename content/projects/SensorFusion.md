Multi-modal sensor fusion project combining YOLOv8 object detection with vehicle telemetry data from the NuScenes autonomous driving dataset. Fuses camera-based object detection with CAN bus telemetry (acceleration, braking) to enrich detection context with driving state information. Includes custom YOLO training on sensor fusion datasets and real-time telemetry extraction pipelines.
