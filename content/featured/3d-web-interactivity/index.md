Built interactive 3D web experiences using Three.js and WebGL, exploring the intersection of data visualization and immersive web technologies. The project includes real-time 3D data visualizations, particle systems, and custom GLSL shaders for rendering complex scientific datasets in the browser.

Features include a physics-based particle simulation engine running entirely on the GPU via compute shaders, interactive 3D graph visualizations for network data, and procedural terrain generation using noise functions. The project demonstrates how modern web technologies can be leveraged to create compelling, interactive scientific visualizations that run at 60fps in the browser.
